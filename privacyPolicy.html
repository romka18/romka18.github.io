Politique de confidentialité

Introduction

    Alloprof est une application de cours en ligne via visioconférence qui permet aux
étudiants de recevoir de l'aide et des conseils de la part de professeurs
expérimentés. Alloprof s'engage à protéger la vie privée de ses utilisateurs et à
respecter les lois applicables en matière de protection des données personnelles.

Données personnelles collectées

Alloprof collecte les données personnelles suivantes de ses utilisateurs :
  • Nom, adresse e-mail, adresse IP.
  • Informations sur les cours suivis et les professeurs consultés
  • Données de navigation

    • Utilisation des données personnelles

    Alloprof utilise les données personnelles de ses utilisateurs pour les finalités suivantes :

  • Fournir le service Alloprof
  • Améliorer le service Alloprof
  • Personnaliser le service Alloprof
  • Envoyer des communications marketing

    • Partage des données personnelles

    Alloprof ne partage les données personnelles de ses utilisateurs qu'avec les personnes suivantes :

  • Les professeurs qui fournissent des cours et des conseils
  • Les partenaires commerciaux d'Alloprof
  • Les autorités compétentes, si nécessaire

    • Droits des utilisateurs

    Les utilisateurs d'Alloprof ont les droits suivants en ce qui concerne leurs données personnelles :

  • Le droit d'accès
  • Le droit de rectification
  • Le droit de suppression
  • Le droit à la portabilité des données

    • Sécurité des données

    Alloprof prend des mesures de sécurité pour protéger les données personnelles de ses utilisateurs, notamment :

  • L'utilisation de technologies de cryptage
  • L'accès restreint aux données personnelles
  • La formation du personnel d'Alloprof

    • Modifications de la politique de confidentialité 

    Alloprof se réserve le droit de modifier sa politique de confidentialité à tout moment.
Les utilisateurs seront informés de toute modification importante de la politique de
confidentialité par e-mail ou par publication sur le site Web d'Alloprof. Alloprof.app
